# 🛍️ Fashion Store React

Application web e-commerce moderne développée avec React.

---

## 🚀 Fonctionnalités
- Affichage des produits de mode
- Ajout au panier dynamique 🛒
- Suppression des articles du panier
- Navigation entre les pages
- Interface utilisateur moderne et responsive
- Animations simples et fluides

---

## 🛠️ Technologies utilisées
- React JS
- JavaScript (ES6+)
- HTML5
- CSS3

---

## 📱 Design
Le projet est entièrement responsive et s’adapte à tous les écrans (mobile, tablette, desktop).

---

## 🎯 Objectif du projet
Créer un site e-commerce simple permettant de simuler une boutique de vêtements en ligne avec gestion du panier.
